# 📰 Fake News Detection using NLP

A Machine Learning based Fake News Detection system that classifies news articles as **Real** or **Fake** using Natural Language Processing and Scikit-learn.

---

## 🚀 Features

* Detects Fake vs Real news
* Text preprocessing and cleaning
* TF-IDF Vectorization
* Machine Learning classification
* Simple and interactive Streamlit UI
* Fast prediction results

---

## 🧠 Machine Learning Model

* Vectorizer: TF-IDF
* Algorithm: Logistic Regression / Linear Model
* Text preprocessing: Lowercasing, punctuation removal, stopword filtering
* Train-test split for evaluation

---

## 🛠️ Tech Stack

* Python
* Scikit-learn
* Pandas
* NumPy
* Streamlit
* Natural Language Processing (NLP)

## ▶️ How to Run Locally

### 1. Clone repository

```
git clone https://github.com/Harshith-b46/Fake-News-Detection-NLP.git
cd Fake-News-Detection-NLP
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the application

```
streamlit run app.py
```

---

## 📊 How It Works

1. User enters news text
2. Text is cleaned and preprocessed
3. TF-IDF vectorizer converts text → numerical features
4. Trained ML model predicts Fake or Real
5. Result displayed in Streamlit UI
